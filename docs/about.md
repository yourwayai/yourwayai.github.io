# 关于 YourwayAI

**YourwayAI** 是一个致力于分享优质开源软件、提升个人与团队生产力的导航站点。我们坚信开源的力量，旨在帮助更多人发现那些能够替代昂贵订阅软件的优秀开源方案。

## 🎯 我们的使命
* **打破信息差**：让更多人知道原本昂贵的工具其实有更好、更自由的开源替代品。
- **简化部署**：为每一个推荐的项目提供最直接的 Docker 部署方案。
- **拥抱社区**：推广优秀的开源文化，支持独立开发者。
